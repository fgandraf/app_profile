FLUXUS WEB API

API Rest desenvolvida para integrar as operações de manipulação e persistência de dados do software Fluxus (gestão de serviços de engenharia para credenciados de bancos públicos) com a base de dados em MySQL.

O projeto está publicado no Microsoft Azure e possui dois tipos de repósitorios de dados. Um real que acessa uma base de dados MySQL e outro fictício (mock) e que pode ser conferido clicando no botão "Projeto" e seguindo as orientações abaixo:
