AEX30 WINDOWS APP

Assistente responsável por automatizar o processo de preenchimento do RAE (Relatório de Acompanhamento de Evolução de Obra) fornecido pela Caixa Econômica Federal, direcionado aos profissionais credenciados no edital de prestação de serviços de arquitetura e engenharia.

Utilizando como base o arquivo de convocação para prestação de serviço e a planilha proposta (PCI) do proponente, o sistema facilita todo o procedimento para o prestador de serviços.

Com uma interface intuitiva, o software guia o usuário passo a passo, garantindo um preenchimento eficiente e preciso. Ao final do processo, o sistema gera a planilha RAE com todos os dados de cabeçalho, percentuais por item, cronograma físico-financeiro e informações contratuais necessárias. A única tarefa restante para o usuário é o preenchimento do apurado em vistoria.
