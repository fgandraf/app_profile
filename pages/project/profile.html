PROFILE WEB PAGE

Projeto web desenvolvido para expor os trabalhos de desenvolvimento de software utilizando HTML, CSS e JavaScript nativos para criar uma experiência rica e interativa.

Para alguém que desenvolvia sites no início dos anos 2000, quando todos os estilos eram aplicados diretamente no arquivo HTML, o Javascript era usado apenas para algumas funcionalidades simples e os efeitos visuais dinâmicos só eram possíveis com tecnologias como o extinto Flash, é surpreendente ver como o desenvolvimento web evoluiu.

Hoje, a possibilidade de aplicar efeitos visuais apenas com CSS, utilizar tags semânticas e desenvolver lógicas de programação mais complexas com Javascript representa uma mudança realmente disruptiva.

Ao longo do tempo, fiz várias modificações, mas tenho mantido o desenvolvimento o mais "vanilla" possível, sem uso de frameworks, explorando ao máximo as funcionalidades nativas da tríade HTML + CSS + JavaScript e uma única biblioteca de terceiros: o SwiperJS.
