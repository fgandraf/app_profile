PROMORE WEB API

API Rest desenvolvida para integrar as operações de manipulação e persistência de dados do software Promore, utilizado para a gestão de vistorias de regularização fundiária pelo Sindicato dos Engenheiros do Estado de São Paulo, com a base de dados em Microsoft SQL Server.

Desenvolvida em .NET 8, esta API faz uso de abstrações fornecidas pelo ORM Entity Framework, incorporando conceitos de boas práticas de código e arquitetura.

O código-fonte da aplicação está disponível em um repositório público no GitHub. Para acessá-lo, clique no botão "Repositório".
